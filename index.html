About

I consider myself a creator and a collaborator, a lover of science and a nerd for technology, a seeker of knowledge and a learner of life, an actor of the stage and a contributor of the arts. The ability to breathe life into an idea, to give form to something that previously did not exist, is one of the most exhilarating feelings for me. Simplify

Live Demo View Code

Simplify is a budgeting application inspired by a zero-based budgeting system. It helps break the process down into 3 easy steps and allows the user to track their spending is specified budget areas of their choosing. The process is flexible enough for the user to use the application in a way that works for them.

Photo Palette

Live Demo View Code

Photo Palette allows users to search the Unsplash photo database for color inspiration. Once a user discovers a photo they find appealing they can have a color palette generated from the photo. The color palette provides the user with the RGB and HEX values of each generated color to be used for future projects.
